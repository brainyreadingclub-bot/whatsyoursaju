# 서비스 개요 — SAJU (What's Your Saju)

---

## 1. 이 서비스가 해결하는 문제

### 5 Whys로 찾은 본질

1. **"사주 서비스를 만들고 싶어"**
   → 왜? 사주에 관심 있는 사람들이 쉽게 접근할 수 있는 서비스가 없어서

2. **"왜 쉽게 접근할 수 없어?"**
   → 기존 사주 서비스는 유료이거나, 단순 띠·별자리 수준에 그쳐 깊이가 없음

3. **"왜 깊이 있는 무료 서비스가 없어?"**
   → 사주명리학은 복잡한 계산(절기경계, 간지, 12운성, 용신 등)이 필요해서 제대로 구현하기 어려움

4. **"왜 제대로 구현이 어려워?"**
   → 전통 사주명리의 계산 체계(입춘 기준 연도, 절기 기준 월주, 장간 가중치 등)를 정확히 이해하고 코드로 옮긴 서비스가 드물기 때문

5. **"그래서 진짜 문제는?"**
   → **사주명리학의 정밀한 계산 체계를 누구나 무료로, 신뢰할 수 있게 접근할 수 있는 서비스가 없다**

### 핵심 문제 정의

> 사주명리에 관심 있는 일반 사용자가 **생년월일시만 입력하면**, 전문가 수준의 정밀한 사주 분석 결과를 **무료로, 즉시, 이해하기 쉽게** 받아볼 수 있는 서비스의 부재

### 확장된 문제 정의

> 나 혼자의 사주를 아는 것을 넘어, **나와 가까운 사람들(가족, 연인, 친구, 동료)의 사주를 함께 비교**하여 서로가 어떻게 보완하고 시너지를 만드는지 알고 싶은 욕구 — 기존 사주 서비스는 1:1 궁합만 제공하고, **3인 이상의 그룹 케미 분석**은 전무

## 2. 서비스의 본질과 핵심

### 한 줄 요약

**생년월일시 입력만으로 전통 사주명리학 기반의 정밀 분석과 그룹 케미 분석을 무료로 제공하는 웹 서비스**

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **정밀성** | 입춘 기준 연도, 절기 기준 월주, 장간 가중치(본기 0.7, 중기 0.3, 여기 0.1)까지 반영한 전통 사주명리 계산 체계 |
| **접근성** | 서버·회원가입·결제 없이 브라우저만으로 즉시 이용 가능 |
| **종합성** | 사주팔자, 대운, 12운성, 오행 분석, 성격, 재물운, 연애운, 건강, 직업운, 올해 운세까지 원스톱 분석 |
| **관계성** | 최대 5인까지 그룹 오행 보완 분석, 서로의 장단점이 어떻게 채워지는지 관계 기반 인사이트 제공 |
| **가독성** | 한자+한글 병기, 오행 색상 시각화, 분석 결과를 일상 언어로 풀어서 전달 |

### 이 서비스가 아닌 것

- 점술·미신을 조장하는 서비스가 아님
- AI가 실시간으로 "상담"해주는 서비스가 아님 (규칙 기반 분석 엔진)
- 유료 전환을 유도하는 프리미엄 서비스가 아님

## 3. 타겟 사용자

### 주 사용자

| 유형 | 특징 |
|------|------|
| **호기심형** | "내 사주가 뭔지 궁금해서" — 가벼운 관심으로 처음 접근하는 2030 세대 |
| **자기이해형** | "나의 성격이나 적성을 다른 관점에서 보고 싶어" — 사주를 자기탐색 도구로 활용 |
| **운세탐색형** | "올해 운이 어떤지 궁금해" — 새해·생일·전환기에 주기적으로 방문 |

### 사용 맥락

- 새해 첫날, 생일, 중요한 결정 앞에서 운세를 확인하고 싶을 때
- SNS에서 사주 관련 콘텐츠를 보고 "나는?" 하며 검색할 때
- 친구·연인끼리 서로의 사주를 비교하며 대화할 때
- **커플/가족/친구 모임에서 "우리끼리 궁합이 어때?" 하며 그룹 케미를 확인할 때**
- **팀 프로젝트/동아리에서 구성원들의 성향과 역할 적합성을 재미로 살펴볼 때**

## 4. 서비스가 만들어지면 달성하는 가치

### 사용자에게

- 전문 사주 카페에서 2~5만원 내고 봐야 할 수준의 분석을 **무료로 즉시** 받아볼 수 있음
- 결과를 **이미지 카드로 저장/공유**하여 소셜 콘텐츠로 활용 가능
- 복잡한 사주명리학 용어를 **일상 언어로 번역**해서 비전문가도 이해 가능

### 서비스 관점에서

- 서버 비용 0원으로 운영 가능한 완전 프론트엔드 아키텍처
- SEO 최적화 + 소셜 공유를 통한 바이럴 성장 가능
- 사주명리 분석 엔진을 검증된 모듈로 확보 → 향후 확장의 기반

## 5. 영감을 받은 유사 서비스

| 서비스 | 특징 | 이 서비스와의 차이 |
|--------|------|-------------------|
| 만세력 앱/사이트 | 간지 테이블만 제공, 해석 없음 | SAJU는 풀이 텍스트까지 자동 생성 |
| 유료 사주 카페 | 전문가 상담, 높은 정확도 | SAJU는 무료+즉시 결과, 접근성 극대화 |
| 별자리/MBTI 서비스 | 가벼운 성격 테스트 | SAJU는 전통 명리학 기반 정밀 계산 |
| 운세 앱 (신한생명 등) | 일일 운세 중심, 얕은 분석 | SAJU는 사주팔자 전체 분석 + 대운 흐름 |

## 6. 현재 구현 상태

### 완성된 기능

- 사주팔자 계산 (년·월·일·시주) — 입춘/절기 경계 반영
- 오행 분석 (장간 가중치 포함)
- 신강/신약 판단 + 용신/희신/기신 산출
- 조후용신 (계절 기반)
- 12운성 (양간 순행, 음간 역행)
- 십신(십성) 분석
- 대운 계산 (순행/역행, 시작나이 자동 산출)
- 육합/상충 감지
- 성격·재물운·연애운·건강·직업운·올해 운세 풀이 텍스트
- 결과 이미지 카드 생성 + URL 공유
- 반응형 UI + 애니메이션

### 미구현/제한 사항

- 음력→양력 변환 미지원 (UI에서 경고 처리)
- 절기일은 평균값 기준 (±1일 오차 가능)
- 시주는 선택 입력 (미입력 시 3주 분석)
- 그룹 궁합 분석 (최대 5인) — **구현 예정**

## 7. 서비스의 미래 상상

### 단기 (현재 → 다음 릴리즈)
- **그룹 궁합/케미 분석 (최대 5인)** — 오행 보완, 장단점 매칭, 그룹 내 역할 분석
- 다양한 출생년도 테스트 케이스 확보

### 중기 (확장)
- 음력→양력 변환 지원
- 절기 정밀 데이터 반영 (연도별 실제 절기일)
- 일진(오늘의 운세) 기능
- 분석 결과 PDF 다운로드

### 장기 (발전)
- LLM 기반 맞춤형 사주 상담 (계산 엔진 + AI 해석)
- 다국어 지원 (영문 사주 풀이)
- 모바일 앱 전환

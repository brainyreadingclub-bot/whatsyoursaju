# 서비스 개요 — SAJU (What's Your Saju)

---

## 1. 이 서비스가 해결하는 문제

### 5 Whys로 찾은 본질

1. **"사주 기반 서비스를 만들고 싶어"**
   → 왜? 사주에 관심 있는 사람들이 제대로 된 해석을 받기 어려워서

2. **"왜 제대로 된 해석을 받기 어려워?"**
   → 기존 무료 서비스는 만세력 계산기 수준이고, 유료 서비스는 1:1 개인 분석에 그침

3. **"왜 1:1 분석에 그쳐?"**
   → 사주의 진짜 가치는 '나'만이 아니라 '나와 주변 사람들의 관계'에서 나오는데, 관계 맥락 분석은 복잡도가 높아 구현이 어려움

4. **"왜 관계 맥락 분석이 어려워?"**
   → N명의 사주를 조합하면 nC2 쌍대 비교 + 그룹 역학 + 시간축 교차 분석이 필요하고, 이를 사람이 이해할 수 있는 해석으로 풀어내려면 명리학 전문성 + AI 해석력이 결합되어야 함

5. **"그래서 진짜 문제는?"**
   → **나를 알고, 내 관계를 읽고, 시기별 처방까지 제공하는 사주 인텔리전스 서비스가 없다**

### 핵심 문제 정의

> 사주명리에 관심 있는 사용자가 **생년월일시만 입력하면**, 전문가 수준의 개인 분석을 즉시 받고, 더 나아가 **주변 사람들과의 관계 역학, 시기별 처방, AI 맞춤 상담**까지 받을 수 있는 서비스의 부재

### 확장된 문제 정의

> 나 혼자의 사주를 아는 것을 넘어, **가까운 사람들(가족, 연인, 친구, 동료)의 사주를 함께 분석**하여:
> - 모든 쌍의 궁합을 매트릭스로 비교 (nC2 조합)
> - 그룹 전체의 오행 균형, 역할 분화, 시너지/갈등 구조를 파악
> - 10년 단위의 대운 교차 타임라인에서 최적의 의사결정 시점을 발견
> - AI 상담으로 구체적 상황에 맞는 개인화된 처방을 받는 것

## 2. 서비스의 본질과 핵심

### 한 줄 요약

**나를 알고, 관계를 읽는 사주 인텔리전스 플랫폼**

### 핵심 가치

| 가치 | 설명 |
|------|------|
| **정밀성** | 입춘 기준 연도, 절기 기준 월주, 장간 가중치(본기 0.7, 중기 0.3, 여기 0.1)까지 반영한 전통 사주명리 계산 체계 |
| **접근성** | 브라우저만으로 가입·설치 없이 즉시 이용 가능. 무료 분석으로 진입 장벽 제로 |
| **종합성** | 사주팔자, 대운, 12운성, 오행, 성격, 재물운, 연애운, 건강, 직업운, 올해 운세까지 원스톱 분석 |
| **관계성** | 2인 궁합을 넘어 최대 5인의 nC2 쌍대 매트릭스 + 그룹 역학 + 시간축 교차 분석 |
| **해석력** | 계산을 넘어 맥락 기반 해석과 시기별 처방. AI(Claude) 기반 개인화 상담 |
| **심층성** | MECE 프레임워크 수준의 체계적 분석: 개인→쌍대→그룹→시간축→실행 인사이트 |
| **가독성** | 한자+한글 병기, 오행 색상 시각화, 전문 용어를 일상 언어로 순화 |
| **지속성** | 1회성 재미가 아닌, 대운 전환기·세운 변동 시 다시 찾아오는 플랫폼 |

### 이 서비스가 아닌 것

- **단순 만세력 계산기가 아님** — 계산을 넘어 해석과 처방을 제공한다
- **1회성 재미 콘텐츠가 아님** — 지속적 관계 기반 인사이트와 시기별 상담을 제공한다
- **전화 상담 대체재가 아님** — AI 기반의 새로운 분석 경험이다 (전문가 매칭 플랫폼이 아님)
- **개인만을 위한 서비스가 아님** — 관계와 집단 역학까지 분석한다
- **점술·미신을 조장하는 서비스가 아님** — 전통 명리학 원리 기반의 자기 이해 도구이다

## 3. 타겟 사용자

### 주 사용자

| 유형 | 특징 |
|------|------|
| **호기심형** | "내 사주가 뭔지 궁금해서" — 가벼운 관심으로 처음 접근하는 2030 세대 |
| **자기이해형** | "나의 성격이나 적성을 다른 관점에서 보고 싶어" — 사주를 자기탐색 도구로 활용 |
| **운세탐색형** | "올해 운이 어떤지 궁금해" — 새해·생일·전환기에 주기적으로 방문 |
| **관계탐색형** | "우리 커플/팀/가족의 케미가 어때?" — 관계 역학을 사주로 이해하고 싶은 사용자 |
| **심층분석형** | "기본 분석을 넘어 구체적 상황에 맞는 처방을 원해" — AI 상담으로 깊은 인사이트를 찾는 사용자 |

### MZ세대 사용 패턴

| 패턴 | 설명 |
|------|------|
| **자기이해 도구** | 사주를 MBTI처럼 "나를 설명하는 렌즈"로 활용 |
| **관계 내비게이션** | "이 사람이랑 맞아?", "상사와 어떻게 지내야 해?" — 관계 질문이 가장 많음 |
| **시즌성 폭발** | 새해/설날(35~40% 집중), 생일, 큰 결정(이직/결혼) 앞에서 급증 |
| **익명성 선호** | 주변에 못 하는 고민을 AI에겐 편하게 — 익명 상담 수요 높음 |
| **구독 피로도** | MZ 평균 3~5개 구독 유지 중 → 연간 패스(월 3,325원)를 주력으로 배치 |
| **가장 많은 질문** | 1. 연애("이 사람이랑 맞아?") 2. 직업("이직 시기") 3. 관계("상사 대응법") |

### 사용 맥락

- 새해 첫날, 생일, 중요한 결정 앞에서 운세를 확인하고 싶을 때
- SNS에서 사주 관련 콘텐츠를 보고 "나는?" 하며 검색할 때
- 친구·연인끼리 서로의 사주를 비교하며 대화할 때
- **커플/가족/팀에서 "우리끼리 궁합이 어때?" 하며 그룹 역학을 확인할 때**
- **기본 분석을 본 뒤 "그래서 올해 이직해야 할까?" 같은 구체적 질문을 AI에게 할 때**
- **대운 전환기가 다가올 때 다시 찾아와 준비 방법을 상담받을 때**

## 4. 수익화 철학

### 핵심 원칙

> **무료는 신뢰 자산이자 최강의 진입 퍼널. 유료는 "왜 그런지 + 어떻게 하라는지"의 대가.**
> **건당 결제가 아닌 구독(관계 인텔리전스 패스)으로 지속적 가치를 제공한다.**

### 가격 모델 (구독 기반)

| 티어 | 가격 | 구성 |
|------|------|------|
| **무료** | 0원 | 개인 사주 전체 분석 + 궁합 점수(숫자+한줄평) + 관계 등록 3명(점수만) + AI 상담 첫 방문 3회(1회성) |
| **월간 패스** | 월 6,900원 | AI 상담 무제한 + 관계 등록 무제한 + 심층 궁합분석(원인+갈등+대응법) + 월운 알림 |
| **연간 패스 (주력)** | 39,900원/년 (월 3,325원) | 월간 전부 + 연간 운세 리포트 + 신년 특별분석 |
| **그룹 분석** | 별도 19,900~39,900원 | 3~5인 nC2 매트릭스 + 팀역학 (구독과 별개 단건 상품) |

### 무료/유료 경계

> **핵심 원칙: 무료는 "뭔지 보여주기", 유료는 "왜 그런지 + 어떻게 하라는지"**

| 무료 (진입 퍼널) | 유료 — 구독 (관계 인텔리전스 패스) |
|-----------------|----------------------------------|
| 개인 사주 전체 분석 (4주+오행+풀이) | AI 상담 무제한 (사주 맥락 기반 처방) |
| 궁합 점수 + 한줄평 ("72점, 상성 좋음") | 심층 궁합 ("午未 육합이 있어서... 갈등 시 이렇게...") |
| 관계 등록 3명 (점수만) | 관계 등록 무제한 + 심층 분석 + 대응법 |
| AI 상담 첫 방문 3회 (1회성 체험) | AI 상담 무제한 |
| - | 월운 알림 + 연간 운세 리포트 (연간 패스) |

### 무료 자산 보호 원칙

- **기존 무료 콘텐츠는 절대 축소하지 않는다**
- 무료 분석의 풍부함이 "이 서비스는 진짜다"라는 신뢰를 만든다
- 유료는 무료를 깎는 것이 아니라, 무료 위에 새로운 층을 쌓는 것이다
- AI 맛보기 3회로 "이건 유용하다"를 체감시킨 뒤 구독으로 연결

## 5. 경쟁 포지셔닝

### 시장 현황

- 한국 점술 시장 규모: 1.4조~4조원
- 주 사용자: MZ세대 80%+, 여성 69~75%
- 1인당 연간 소비: 약 8만원 (10~39세)
- 모바일 이용 42.6%, 오프라인 25.2%

### 경쟁사 비교

| 서비스 | 매출 | 핵심 모델 | 우리와의 차이 |
|--------|------|-----------|-------------|
| **점신** | 978억원 (테크랩스) | 전문가 전화상담 수수료 | 사람 상담 vs AI 상담 |
| **포스텔러** | 509억원 | 캐릭터 시각화 + 인앱구매 | 재미/캐릭터 vs 관계 인텔리전스 |
| **헬로우봇** | 월 1.2억원 | 챗봇 타로 + 월 9,900원 구독 | 타로 챗봇 vs 명리학 기반 AI (우리 6,900원 vs 헬로우봇 9,900원) |
| **척척사주** | 미공개 | AI 사주풀이 건당 14,800원 | 개인 분석 vs 관계+그룹 분석 |

### 우리의 고유 영역

> **"AI 기반 관계 사주 인텔리전스"** — 전화상담(점신)도 캐릭터 운세(포스텔러)도 아닌,
> 나와 내 주변 사람들의 관계를 체계적으로 분석하는 새로운 카테고리

## 6. 서비스가 만들어지면 달성하는 가치

### 사용자에게

- 전문 사주 카페에서 5~50만원 내고 봐야 할 수준의 분석을 **합리적 가격에 즉시** 받음
- 개인 분석은 **무료로 충분히 깊은 수준**을 제공
- AI 상담으로 **"그래서 나는 어떻게 해야 해?"**라는 질문에 구체적 답변을 받음
- 그룹 분석으로 **팀/가족/커플의 역학을 이해**하고 관계를 개선
- 결과를 **이미지 카드로 저장/공유**하여 소셜 콘텐츠로 활용

### 서비스 관점에서

- 무료 분석 → AI 상담 → 그룹 분석의 **자연스러운 전환 퍼널**로 수익 창출
- 무료 기능은 서버 비용 0원 유지 (프론트엔드 전용)
- 유료 기능은 Vercel 서버리스 + Claude API로 **종량제 운영** (고정비 최소)
- SEO + 바이럴 공유를 통한 유기적 성장

## 8. 현재 구현 상태

### 완성된 기능

- 사주팔자 계산 (년·월·일·시주) — 입춘/절기 경계 반영
- 음력→양력 변환 (korean-lunar-calendar CDN)
- 오행 분석 (장간 가중치 포함)
- 신강/신약 판단 + 용신/희신/기신 산출 + 조후용신
- 12운성 (양간 순행, 음간 역행) + 십신(십성)
- 대운 계산 (순행/역행, 시작나이 자동 산출)
- 합충형파해 감지 (육합/삼합/상충/삼형/파/해)
- 신살 6종 (도화/역마/화개/천을귀인/천덕/월덕) + 공망
- 세운 3년 흐름 + 분기별 포인트
- 성격·재물운·연애운·건강·직업운·올해 운세 풀이 텍스트
- 궁합 미니게임 (2인 간이 궁합 — 점수+등급+케미)
- 사주유형 20종 판별 (정관격, 편관격, 식신생재 등)
- 결과 이미지 카드 생성 + 카드 공유 버튼 + URL/Web Share 공유
- 대운 신강/신약 분기별 해석
- 연애스타일 스코프 수정 (일간+용신+합충 기반)
- 3-Tier 결과 UI + 탭 기반 상세 분석
- 프리미엄 블러 카드 + 가격표 + Formspree 이메일 수집
- 모바일 sticky CTA bar + 반응형 UI + 로딩 애니메이션
- 83개 계산 테스트 통과 (test_step1~3, test_review)

### 미구현 — 다음 Phase에서 구현

- 결제 시스템 (PortOne 연동)
- AI 맞춤 상담 (Claude API)
- 프리미엄 2인 심층 궁합
- 그룹 분석 (3~5인 nC2 매트릭스 + 팀 역학)
- 시간축 분석 (10년 타임라인, 황금 교차점)
- 월운 캘린더 (신규 계산 함수 개발 필요)
- 사용자 계정 시스템
- MECE 풀 레포트 PDF 생성
